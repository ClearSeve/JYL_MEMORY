# 编程笔记

这玩意是我从2014毕业那年就开始写的，包含了很多内容。

里面我写得很精炼，连图片都没有插入。

因为时间跨度大，我写笔记的精简度也不断提高，刚开始写的内容还包含了我对概念的理解，后面基本我全都是只写了功能和知识点的罗列，没有任何描述性的东西。

这个笔记我主要用来在写代码的时候进行功能查询~
