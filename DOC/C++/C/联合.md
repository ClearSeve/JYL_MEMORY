# 联合

和结构体用法相同(关键字用union)  
所有成员共用起始地址相同的一片空间
 
union A{int m;char data[4];}  
m=0x 44 43 42 41   
输出data的四个元素为4个十六进制数的ASCII码代表的字符  
int占4个字节,1个字节8位。8个1代表数字255  
char占1个字节,该内存大小可存8位二进制，2位十六进制，int相当于4个char  
m: 44(data[0])   43   42   41(data[3])

大端模式：低字节存高位数据
小端模式: 低字节存低位数据

```
union {
	int i;
	char x[2];
}a;
int main()
{
	a.x[0] = 10;  //0A 
	a.x[1] = 1;   //01
	int ret = a.i;//0x010A    266
}
```
