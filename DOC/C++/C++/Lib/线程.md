## 原子类型

线程等待
```
atomic_bool sig(false);
//atomic_char、	atomic_uint、atomic_ulong


//等待设置true后执行
while(1)
{
    while (!sig);
    。。。。
    sig = false;
}


//命令执行
sig = true;
```
