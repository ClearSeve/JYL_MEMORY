# 特性

```
c++头文件无.h,需要使用c语言头文件时：
<stdio.h>或<cstdio>
std命名空间有：cin，cout，endl，string，fstream等
c++后缀为：.cpp   .C   .cxx

赋值可以用：	int i(3);

c++中结构、联合、枚举
struct、union、enum不再是类型的一部分，名字就是类型名
struct  A{......};    A  a;
c++允许使用匿名联合

c++的struct可以像class那样用，不同的是结构体默认是public，而class默认是private。
没有任何字段的结构体，c++大小1，c中为0
结构体封装数据，函数封装代码，结构体大小与定义的函数数量无关，只与变量有关。
```
