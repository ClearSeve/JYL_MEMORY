# 对象选择

## 元素选择器

```
$("#abc")  选取 id='abc'的元素
$("p") 选取 <p> 元素
$("p.intro") 选取所有 class="intro" 的 <p> 元素
$("p#demo") 选取所有 id="demo" 的 <p> 元素
$("p[xx='#']") 选取所有 xx="#" 的 <p> 元素
```

## 属性选择器

```
$("[xx]") 选取所有带有 xx属性的元素
$("[xx='#']") 选取所有带有 xx值等于 "#" 的元素
$("[xx!='#']") 选取所有带有 xx值不等于 "#" 的元素
$("[xx$='.jpg']") 选取所有 xx值以 ".jpg" 结尾的元素
```

## CSS选择器

```
$("p").css("background-color","red");
所有 p 元素的背景颜色更改为红色
```

## 连续操作

$("p").css("color","red").slideUp(2000).slideDown(2000);
