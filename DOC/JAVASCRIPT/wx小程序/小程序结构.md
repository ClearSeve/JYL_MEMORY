## app.json

全局配置，pages中定义所有页面路径  
字段的第一个页面为首页

## 页面

每个页面必须要有对应的js文件，文件必须包含Page({})对象
每个页面中的.json可单独定义该页面的样式

```
Page({
  data: {
    dat1: 'xx',
    dat2: {}
  },

  Fun1: function() {
  }

  onLoad: function () {//页面载入事件
    console.log("xxx");
  }
  onShow: function () {//页面显示事件
  }

})

访问数据：data.dat1
设置数据：this.setData({ dat1: "xxxx" })
```
