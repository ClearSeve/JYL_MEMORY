## 读写utf8

```
import codecs

ll = codecs.open(r"D:\a.txt", 'r', 'utf-8').readlines()
of = codecs.open(r"D:\a.txt", 'w', encoding='utf8')
for line in ll:
    of.write(line)
of.close()
```
