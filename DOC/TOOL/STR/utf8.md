# utf8
根据每个字节前面的数字决定后面使用几个字节表示一个unicode编码

```
0-0x7F             0*******
0x80-0x07FF        110***** 10******
0x800-0xFFFF       1110**** 10******  10******
0x10000-0x10FFFF   11110*** 10******  10******  10******

‘汉’的unicode编码
十六进制：6C49
二进制：01101100 01001001

二进制分割为：    0110    110001    001001
套入模板：    1110****  10******  10******
对应utf编码为 11100110  10110001  10001001
```
