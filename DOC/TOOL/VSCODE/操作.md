# 操作

插件安装路径      C:\Users\用户名\.vscode  

## 快捷键

+ ctrl+shift+`     打开终端
+ F1               命令输入
+ F11              全屏
+ shift+alt+左键    竖排文本选择  
+ ctrl +           界面放大  
+ ctrl -           界面缩小
+ ctrl+k v         调出预览窗口

输入!,按Tab，生成html代码
