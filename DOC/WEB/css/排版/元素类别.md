# 元素类别

## 块元素
```
独占一行，其后的元素另起一行显示,width、height、padding、margin都可控

div,p,h1,h2,h3,h4,h5,h6,table,ul,li,ol,form
address,blockquote,center,dir,dl,fieldset,hr,isindex,menu,noframes,noscript,pre 
```

## 内联元素
```
和相邻内联元素在同一行,WrapPanel方式排版
width、height、padding-top/padding-bottom、margin-top/margin-bottom不可变
内部文字或图片的大小可变

label,br,img,input,span,a,textarea,code
abbr,acronym,b,bdo,big,cite, dfn,em,font,i,kbd,q,s,samp,select,small,strike,strong,sub,sup,tt,u,var
```

## 可变元素
```
根据上下文关系确定该元素是块元素还是内联元素

button,iframe,script
del,ins,map,object
```
