# 右键菜单

```
<控件类型.ContextMenu>
        <ContextMenu Name="cm" StaysOpen="true">
                    <MenuItem Header="选择文件夹"/>
                    <MenuItem Header="一级菜单">
                        <MenuItem Header="二级菜单"/>
                    </MenuItem>
        </ContextMenu>
 </控件类型.ContextMenu>
```
