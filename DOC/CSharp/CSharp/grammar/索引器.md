# 索引器

使类可以通过[]语法访问和赋值。

```
class Test
{
    string Content1 = "aaa";
    string Content2 = "bbb";
    public string this[int index]
    {
        get
        {
        if(index == 1) return Content1;
        if (index == 2) return Content2;
        return "erro";
        }
        set { }
    }
}

Test test = new Test();
Console.WriteLine(test[1]);
```
