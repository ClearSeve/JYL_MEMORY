## 接口
```

type INTER interface {
	fun()
}

type STU1 struct {
}
func (val STU1) fun() {
	fmt.Println("STU1")
}


type STU2 struct {
}
func (val STU2) fun() {
	fmt.Println("STU2")
}


var inter INTER

inter = new(STU1)
inter.fun()

inter = new(STU2)
inter.fun()
```

## 数组

类型:  
[4]int

定义:  
var n [4]int = [4]int{1, 2, 3, 4}

切片:  
var a = n[2:]  //取第三个元素到末尾  
var b = n[:2]  //取头到第二个元素  
var c = n[1:3] //取第二到第三个元素

遍历:
```
for num := range n {
	fmt.Println(num)
}
```

## map
