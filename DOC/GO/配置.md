## 环境变量

PATH包含go的bin目录  

GOPATH下存放库(默认位置为C:\Users\jyl\go)
(github库下载go get github.com/...)
(存储目录GOPATH下src)


vi /etc/profile
添加：
export PATH=go的bin目录:$PATH  
export GOPATH=包存放目录

source /etc/profile  


## 运行代码  
go run XX.go
