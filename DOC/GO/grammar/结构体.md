# 结构体

结构体都继承自interface{}

```
//People people
type People struct {
   name string
   age int
}
func (_this *People) Init(val int) {
	_this.age = val
   fmt.Println(_this.name)
}

aa := People{"xx", 100}
aa.Init(10)

cc := People{age:100}

var bb = People{}
bb.name = "yy"
pa = &aa
fmt.Println(pa.name)

pdat := new(People)
```
