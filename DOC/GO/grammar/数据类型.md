### 变量声明
```
var x = 10                 //var 变量名 = 初始值

var a int = 0              //var 变量名 类型 = 初始值
var b, c int = 1, 2

newval := 10               //新变量名 := 初始值
newval, newval2 := 10, 20  //必须包含未声明的新的变量
```
