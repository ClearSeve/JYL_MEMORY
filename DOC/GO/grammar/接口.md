# 接口

## 定义接口

```
//INTER 接口类型(指向结构体)
type INTER interface {
	fun1(val int)
}
```

## 实现接口

```
//STU  可用转换为接口的结构体（分别实现了接口中的函数）
type STU struct {
	x int
}

func (_this *STU) fun1(val int) {
	_this.x += val
	_this.fun2()
}
func (_this *STU) fun2() {
	_this.x += 100
}

```

## 调用

```
stu := STU{1}
stu.fun1(1)
fmt.Println(stu.x) //102

var ii INTER = &stu
ii.fun1(10)
fmt.Println(stu.x) //212

```
