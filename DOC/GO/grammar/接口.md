# 接口

## 定义接口

```
//INTER 接口类型
type INTER interface {
	fun1(val int)
}
```

## 实现接口

```
//STU  可用转换为接口的结构体（分别实现了接口中的函数）
type STU struct {
	x int
}

func (_this *STU) fun1(val int) {
	_this.x += val
	_this.fun2()
}
func (_this *STU) fun2() {
	_this.x += 100
}

```

## 调用

```
stu := STU{1}
stu.fun1(1)
fmt.Println(stu.x) //102

var ii INTER = &stu//函数前声明的结构体类型都是指针，转换接口类型就用指针
ii.fun1(10)
fmt.Println(stu.x) //212

var i2 *INTER = &ii
(*i2).fun1(10)//接口必须使用对象类型访问
```
